# EZLabs Contact Form

React single-page app for EZ Labs assignment.

## Run locally
1. Install dependencies:
\`\`\`
npm install
\`\`\`
2. Start dev server:
\`\`\`
npm run dev
\`\`\`
3. Open http://localhost:5173
